System Overview

Dual Portal System

Separate secure portals for students and academic advisors with role-based access control

2

Smart Registration

Real-time validation against prerequisites, locked courses, and graduation eligibility

3

AI-Powered Advisor

RAG-based chatbot trained on Helwan University's 2021 Credit Hour Regulations

4

Zero Infrastructure

Streamlit web interface with CSV persistence - no database or server setup required

✨ Key Capabilities

🎓 Student Experience

  • Secure Login: Student ID + name-based verification code
  • Graduation Protection: Blocks Year 4 "Ready to Graduate" students
  • Multi-Course Registration: Select multiple courses with real-time validation
  • Request Management: View, edit, or delete pending requests

👨‍🏫 Advisor Tools

  • Analytics Dashboard: CGPA distribution, payment status, attendance risk
  • Request Workflow: One-click approval/rejection with justification
  • Risk Detection: Automated identification of at-risk students
  • Data Export: CSV export with custom column selection

🤖 AI Integration

  • RAG Chatbot: Answers academic questions using official regulations
  • Regulation Compliance: Trained on Decision No. 3257/2021
  • Context-Aware: Provides precise answers with source references
  • Arabic Support: Full Arabic language processing capabilities

🛡️Regulatory Compliance

🔐 Data Isolation

Students can ONLY access their own requests. Advisors can ONLY manage their assigned students.

📜 Ministry Compliance

Fully compliant with Helwan University Credit Hour System (Decision No. 3257/2021).

💾 Persistent Audit Trail

All actions timestamped in CSV files with modification tracking and status history.

⚙️ Technical Implementation

Frontend

Streamlit - Responsive web interface with accessibility features (large fonts, high contrast)

Backend

Modular Python - Separate systems for student registration and advisor management

AI Engine

RAG Pipeline - SentenceTransformer embeddings + FAISS indexing + LLM integration

Data Layer

CSV Persistence - Human-readable format compatible with existing university systems
